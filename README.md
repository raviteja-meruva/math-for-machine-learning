# Mathematics for Machine Learning

This repository documents my study of **mathematical foundations essential for machine learning**,
with a focus on **intuition, clarity, and practical relevance** rather than heavy formal proofs.

The goal of this repository is to:
- Understand *why* machine learning algorithms work
- Build intuition behind model behavior and optimization
- Strengthen foundations required for ML, Deep Learning, and AI systems

This repository complements my Python and ML projects and evolves gradually alongside them.
